
# <font face="monospace">HH STAY 票券電商</font>
使用 Vite 串接 API 完成的單頁式應用旅遊套票電商網站，包含 RWD 響應式體驗
提供產品瀏覽、分類篩選、價格排序、加入購物車及結帳付款等功能

![](https://i.imgur.com/veygUJd.jpg)
![](https://i.imgur.com/Ngusacg.png)
